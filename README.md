# 🚀 TaskFlow Pro - Gestão de Fluxo Empresa X

O **TaskFlow Pro** é uma aplicação web minimalista e eficiente desenvolvida para organizar o fluxo de trabalho diário. Permite a gestão de tarefas com diferentes níveis de prioridade e prazos, garantindo foco no que é realmente crítico.

## 🛠️ Tecnologias Utilizadas

* **HTML5 Semântico**: Estrutura acessível e organizada.
* **CSS3 Moderno**: Layout responsivo com Variáveis, Flexbox e Grid.
* **JavaScript (ES6+)**: Lógica dinâmica e manipulação do DOM.
* **LocalStorage**: Persistência de dados diretamente no navegador.

## ✨ Funcionalidades

- **Adição de Tarefas**: Título, descrição detalhada, prioridade e prazo.
- **Priorização Visual**: Barras laterais coloridas indicando o nível de urgência (1 a 4).
- **Abas de Filtragem**: Alterna entre ver todas as tarefas, apenas pendentes ou concluídas.
- **Persistência**: As tuas tarefas não desaparecem ao atualizar a página.
- **Responsivo**: Funciona em computadores, tablets e telemóveis.

## 🚀 Como Testar

1. Faz o download ou clona este repositório.
2. Abre o ficheiro `index.html` em qualquer navegador moderno.
